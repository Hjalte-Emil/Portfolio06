For at køre projektet.

1. Opret databasen:
   Åben portfolio6_database.sql med MySQL Workbench og kør filen.

2. Installer afhængige packages:
   Åben projektet og i terminalen kør: npm install
   (Den finder selv frem til de korrekte packages, jævnfør packages.json)

4. Ændr connection password "MySQL Password" på linje 15 i server.js

5. Tænd for server.js:
   Klik på den grønne knap i toppen af WebStorm eller i terminalen skriv: npm start
